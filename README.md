# 🛡 PhishGuard AI – Intelligent Phishing Detection & Threat Analysis

**PhishGuard AI is a real-time cybersecurity intelligence system that detects, explains, and simulates phishing attacks using advanced AI.**

> Built for Tech Builder Program Hackathon 2026 — Cybersecurity & Privacy Track

## 🚀 What It Does

PhishGuard goes beyond simple phishing detection.

Instead of just saying “safe” or “phishing”, it:
- 🧠 **Analyzes attacker intent**
- 🔍 **Breaks down technical signals**
- 🧬 **Reconstructs the attack flow**
- 💡 **Educates users on why it matters**

Paste any:
- URL
- Email
- Message / SMS

…and get a full cybersecurity analysis instantly.
